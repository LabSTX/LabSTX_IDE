# LabSTX Release v1.2.0

We are excited to announce LabSTX v1.2.0! This release focuses on stability, multi-file support, and project sustainability.

## 🚀 Key Features

### 1. Multi-Contract Support
LabSTX now fully supports multi-contract project architectures. Developers can add multiple `.clar` files to their workspace, with the integrated Clarinet runtime automatically managing dependencies and contract calls.

### 2. Early Access Program
The official "Join Early Access" program is now live! We've integrated dedicated calls-to-action in the AI assistant's empty state, the main dashboard, and the project documentation to help users shape the future of browser-based Clarity development.

### 3. Stability & Performance
- **WebContainer Fix**: Resolved the "hanging terminal" issue and stabilized WebContainer loading in production environments.
- **Editor Optimization**: Fixed a cursor-jumping bug where the editor would incorrectly snap to the last error during high-speed typing.

## 📜 Documentation & Legal
- **MIT License**: LabSTX is now officially licensed under the MIT License.
- **Enhanced README**: We've updated our documentation with a comprehensive comparison table against Hiro Playground and Claride to help developers choose the best tool for their workflow.
